# Dharun V — Portfolio

A unique developer portfolio built with HTML, CSS & Node.js.

## Features
- Animated particle + grid canvas hero
- Custom cursor with smooth follower
- Terminal typewriter animation
- Scroll-reveal sections
- Infinite skills marquee
- Working contact form (Node.js backend)
- Sticky blur navbar
- Fully responsive

## Setup

```bash
# Install dependencies
npm install

# Run in development (with auto-restart)
npm run dev

# Run in production
npm start
```

Open http://localhost:3000

## Customise

1. **Your info** — Edit `public/index.html`:
   - Replace college name, internship, club details
   - Update project names, descriptions, tech stacks
   - Update contact links (email, GitHub, LinkedIn)

2. **Contact form** — Edit `server.js` to add nodemailer for real email delivery

3. **Deploy** — Push to GitHub and deploy on Railway, Render, or Vercel
